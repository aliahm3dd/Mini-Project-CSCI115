# Arab Restaurant Website 

##  Description

A multi-page restaurant website inspired by Middle Eastern cuisine.
This project includes organized sections for menu categories like starters, main course, and desserts, with a clean and structured layout.

---

##  Features

* Multi-page navigation (Home, About, Menu)
* Separate pages for:

  * Starters
  * Main Course
  * Desserts
* Styled food menu cards
* Simple and clean UI
* Basic interactivity on the Home page using JavaScript

---

##  Technologies Used

* Visual Studio Code
